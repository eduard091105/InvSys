# Assets do InvSys

Coloque aqui os arquivos visuais do sistema.

Para o executavel e o instalador usarem um icone automaticamente no Jenkins, salve o arquivo como:

```text
assets/app.ico
```

Recomendado:

- formato `.ico`
- tamanhos internos 16x16, 32x32, 48x48 e 256x256
- fundo transparente, se for logo

Se voce tiver apenas PNG/JPG, envie a imagem para mim e eu posso converter/preparar o icone para `assets/app.ico`.
